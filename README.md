# Chat Room Project

A simple real-time chat room application built with Python using Socket programming and multithreading.

## Features
- Multi-client support
- Unique nicknames for each user
- Real-time message broadcasting to all connected clients
- Handles client disconnections gracefully

## How to Run
1. Start the server: `python server.py`
2. Run multiple clients: `python client.py`
3. Enter your nickname and start chatting!

## Technologies Used
- Python
- Sockets
- Threading
